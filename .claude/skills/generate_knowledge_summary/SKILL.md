# generate_knowledge_summary - 知识摘要生成器

## 🎯 核心功能

从知识内容中提取关键信息，识别核心要点，构建清晰的摘要结构，优化表达的简洁性，并验证信息的完整性。

## 📋 工作流SOP

**工作流SOP**：
```
1. 接收知识内容
2. 提取关键信息
3. 识别核心要点
4. 构建摘要结构
5. 优化表达简洁性
6. 验证信息完整性
7. 返回知识摘要
```

### 详细流程说明

**步骤1：接收知识内容**
- 验证输入内容的完整性和有效性
- 检查摘要长度要求的合理性
- 解析目标读者和重点强调参数
- 记录内容基本信息

**步骤2：提取关键信息**
- 识别文档中的关键句子和段落
- 提取重要的事实和数据
- 识别核心概念和定义
- 生成关键信息候选集合

**步骤3：识别核心要点**
- 分析文档的主题结构
- 识别主要论点和支撑论据
- 提取关键结论和建议
- 生成核心要点列表

**步骤4：构建摘要结构**
- 设计摘要的逻辑结构
- 组织要点之间的层次关系
- 确定摘要的篇幅分配
- 生成结构化摘要框架

**步骤5：优化表达简洁性**
- 精简语言表达，去除冗余
- 使用简洁明了的句式
- 保持专业性和准确性
- 优化摘要的可读性

**步骤6：验证信息完整性**
- 检查关键信息的覆盖度
- 验证摘要与原文的一致性
- 评估信息保留的完整性
- 计算信息覆盖百分比

**步骤7：返回知识摘要**
- 生成结构化的摘要内容
- 包含关键要点列表
- 提供原文覆盖度百分比
- 返回可读性评分

## 🎯 适用场景

- **快速预览**: 为长文档生成执行摘要供快速阅读
- **知识传播**: 将复杂政策简化为易懂的要点
- **报告生成**: 从调研材料中提取关键发现
- **内容提炼**: 将详细说明浓缩为核心要点

## 🚀 快速开始

```
知识内容：年假政策完整文档
```

