# Skill Factory - 智能Skill生成器

## 🎯 概述

`skill_factory` 是一个标准化的Skill生成系统，基于预定义的模板和用户配置参数，生成符合规范的完整技能包。无论是初学者还是经验丰富的开发者，都能快速创建高质量、标准化的Skill。

## 🚀 快速开始

### 触发方式

**方式1：明确请求**
```
请帮我创建一个新的Skill
或者
使用skill_factory创建数据处理Skill
```

**方式2：场景识别**
当检测到以下情况时自动建议：
- 用户重复执行相似复杂操作
- 现有Skill组合无法高效解决问题
- 用户表达"希望自动化"某类任务

### 创建流程

#### 第一阶段：需求收集（对话式）
1. **基本信息收集**
   - Skill名称和功能描述
   - 目标场景和问题定义

2. **功能细节深化**
   - 根据Skill类型针对性询问
   - 渐进式细化功能规格

3. **技术需求确认**
   - 确认是否需要脚本、模板、参考文档
   - 确认目标用户技术水平

#### 第二阶段：内容生成（自动化）
1. **智能模板选择** - 基于Skill类型和复杂度选择最佳模板
2. **分层文档生成** - 根据用户水平调整技术细节
3. **质量验证流水线** - 确保生成内容符合标准
4. **预览确认** - 提供完整预览供用户修改

#### 第三阶段：交付部署
1. **文件写入** - 一次性生成完整Skill包
2. **使用指导** - 提供启用和使用说明
3. **后续支持** - 提供自定义扩展指导

## 📋 输入规范

### 必需输入
- **skill_name**: Skill名称（小写下划线格式）
- **description**: 功能描述（清晰明确）
- **skill_type**: Skill类型（data/api/doc/file/content）

### 可选输入
- **complexity**: 复杂度级别（simple/medium/complex）
- **target_audience**: 目标用户（beginner/intermediate/expert）
- **include_scripts**: 是否包含脚本（true/false）
- **include_templates**: 是否包含模板（true/false）
- **custom_requirements**: 自定义需求

## 📤 输出内容

### 标准输出结构
```
.claude/skills/<skill_name>/
├── SKILL.md              # 主技能文件（符合YAML规范）

└── examples/             # 使用示例（如需要）
```

## 🎪 使用示例

### 示例1：创建数据处理Skill
```
用户：我想创建一个处理Excel数据的Skill

Skill Factory：
1. 询问：Skill的主要功能是什么？（数据清洗/格式转换/数据分析）
2. 询问：处理哪些Excel格式？（xlsx/csv/tsv）
3. 询问：需要哪些特定功能？（公式计算/图表生成/数据验证）
4. 生成：完整的Excel数据处理Skill包

Skill Factory：
1. 询问：API端点URL和认证方式

3. 询问：错误处理策略
4. 生成：完整的API集成Skill包
```

## 🔧 高级功能

### 模板库支持
- **标准模板**：5种预设模板类型
- **自定义模板**：支持用户保存个性化模板
- **批量生成**：基于模板批量创建相似Skill

### 版本管理
- 记录Skill生成历史和修改记录
- 支持版本回滚和比较
- 模板版本控制

### 智能推荐
- 基于使用历史推荐相似模板
- 根据问题复杂度推荐最佳实践
- 提供Skill组合建议

## ⚠️ 注意事项

### 适用范围
✅ **支持类型**：
- 数据处理和转换
- 文档生成和格式化
- 文件操作和管理
- 简单API集成
- 内容创作辅助

❌ **限制类型**：
- 复杂系统集成
- 高性能计算任务
- 安全敏感操作
- 需要专门开发的功能

### 最佳实践
1. **命名规范**：使用小写下划线格式（如 `data_processor`）
2. **功能单一**：每个Skill专注于一个明确的功能
3. **文档完整**：确保README包含清晰的使用说明
4. **错误处理**：在脚本中包含适当的异常处理
5. **依赖管理**：明确列出所有依赖库和版本要求

## 🔄 迭代优化

支持用户对生成的Skill进行修改和优化：
- **内容调整**：修改SKILL.md和文档内容

- **模板更新**：基于使用反馈优化模板库
